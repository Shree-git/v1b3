# Active check modules (v2.0)

check_tls()                  # cert expiry, SSL errors
check_security_headers()     # HSTS, CSP, X-Frame-Options, X-Content-Type-Options,
                             #  Referrer-Policy, Permissions-Policy
check_cors()                 # reflected origin + ACAC:true only
                             # ACAO:* alone is NOT flagged (correct for public APIs)
check_graphql()              # introspection query to /graphql, /api/graphql, /gql
check_cookies()              # HttpOnly, Secure, SameSite flags
check_path()                 # 40+ sensitive paths (.env, .git, SQL dumps, wp-config...)
check_debug_endpoints()      # Spring Boot Actuator, Prometheus, Swagger UI,
                             #  H2 console, phpMyAdmin, WireMock, Apache server-status
check_directory_listing()    # "Index of /" pattern in /static/, /uploads/, /files/...
extract_js_urls()            # parses all <script src> and chunk references
scan_for_secrets()           # 22 patterns: OpenAI, Stripe, AWS, GitHub, Slack,
                             #  Google, Supabase, SendGrid, Mailgun, DB connection strings
check_source_maps()          # .js.map files — reconstructs unminified source
check_prototype_pollution()  # __proto__[], Object.assign({}, req.body), lodash.merge
get_spa_fingerprint()        # canary request to /v1b3-canary-xyz987654321-nonexistent
                             # filters SPA catch-all 200s from real hits
severity.py — scoring logic
# Severity tiers

CRITICAL  # Secrets in code, heap dumps, DB admin panels,
          # Database connection strings with credentials,
          # Private key material

HIGH      # Env files exposed, .git config, Spring Boot /actuator/env,
          # CORS reflected-origin + ACAC:true, source maps,
          # TLS cert expiring <14 days, missing HSTS

MEDIUM    # GraphQL introspection, Swagger UI, missing CSP,
          # X-Frame-Options absent, debug endpoints,
          # TLS cert expiring <30 days

LOW       # Missing X-Content-Type-Options, Referrer-Policy,
          # Permissions-Policy, insecure cookie flags,
          # Prototype pollution patterns

INFO      # Health endpoints, robots.txt, version disclosure

DECISIONS.md — key architectural choices
2026-02-20  CORS detection: Only flag reflected-origin + ACAC:true combo.
            ACAO:* wildcard alone is correct behavior for public APIs.
            (Filed + retracted context7.com false positive. Scanner fixed.)

2026-02-20  SPA false-positive filtering: Canary request to random nonexistent
            path. If it returns 200 + HTML → server serves index.html for
            all paths → filter all path hits against this fingerprint.

2026-02-20  Secret redaction: Show first 8 chars of matched secret + "...".
            Never log or store full values. Confirm type, redact value.

2026-02-20  Target scope: Vibe-coded/AI-generated apps preferred over mature
            production infrastructure. Higher signal-to-noise ratio,
            more likely to have config mistakes, more receptive owners.

2026-02-20  Blog architecture: HTML + vanilla JS over Quarto/Jekyll.
            No build step, no dependencies, pure GitHub Pages static.
            Auto-publish via Python blog_writer.py + git push.
